# Еженедельный мониторинг перегруза

Прототип веб-приложения для отслеживания сверхурочной работы: сводные метрики, фильтры, таблица специалистов со статусами и приоритетом отдыха.

## Стек

- React 18 + TypeScript
- Vite
- Recharts, Lucide React

## Запуск

```bash
npm install
npm run dev
```

Сборка: `npm run build`. Тесты: `npm run test`.
